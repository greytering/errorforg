# How to create A file to github?
Setp 1: touch a file
```
# touch Filename
Example: touch file
```
Setp 2: Add to stage
```
# git add Filename
Example: git add file
```
Setp 3: Submit changes
```
# git commit -m "Information"
Example: commit -m "Update"
```
Setp 4: Update to github
```
# git push origin master
Example: git push origin master
```


# Example
```
touch test
git add test
git commit -m "Upload"
git push origin master
```
