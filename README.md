# How to create A file to github?
Setp 1: touch a file
```
$ touch Filename
```
Setp 2: Add to stage
```
$ git add Filename
```
Setp 3: Submit changes
```
$ git commit -m "Information"
```
Setp 4: Update to github
```
$ git push origin master
```


# Example
```
touch test
git add test
git commit -m "Upload"
git push origin master
```
